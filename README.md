# Pong Game

Un jeu Pong simple créé avec Pygame.

## Description

Ce jeu est une implémentation classique de Pong où deux joueurs contrôlent des raquettes pour frapper une balle.

## Fonctionnalités

- Jeu à deux joueurs
- Système de score
- Détection de collision
- Écran de victoire

## Installation

```bash
Cloner le dépôt
Installer pygame
pip install pygame
```

## Comment jouer

```bash
python pong.py
```

### Contrôles
- Joueur gauche : Z (haut), W (bas)
- Joueur droit : Flèche haut, Flèche bas

## Prérequis

- Python 3.12+
- Pygame 2.6.1+
